# Block Drop

DropBlock is an all Python game using **Pygame**, inspired by the classic Tetris by Alexey Pajitnov.

---
## Controls
| Action           | Key         | Controller                |
|------------------|-------------|---------------------------|
| Move Left        | Left Arrow  | DPad or Joystick Left     |
| Move Right       | Right Arrow | DPad or Joystick Right    |
| Drop             | Down Arrow  | Dpad Down                 |
| Rotate           | Up Arrow    | X, A, B (Buttons 0, 1, 3) |
| Pause / Unpause  | Space       | + (Button 7)              |
| Restart Game     | R           | - (Button 6)              |

---

## Launching the game  
  
Inside the dist folder, launch `DropBlock.exe`  
If you wish to run the game via python, ensure you have Pygame and run via `src/DropBlock.py`
